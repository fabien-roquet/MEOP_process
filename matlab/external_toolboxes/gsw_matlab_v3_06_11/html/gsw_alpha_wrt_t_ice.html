USAGE:

gsw_alpha_wrt_t_ice = gsw_alpha_wrt_t_ice(t,p)

DESCRIPTION:

Calculates the thermal expansion coefficient of ice with respect to
in-situ temperature.

INPUT:

t   =  in-situ temperature (ITS-90)                            [ deg C ]
p   =  sea pressure                                             [ dbar ]
       ( i.e. absolute pressure - 10.1325 dbar )
p may have dimensions 1x1 or Mx1 or 1xN or MxN, where t is MxN.

OUTPUT:

alpha_wrt_t_ice  =  thermal expansion coefficient of ice         [ 1/K ]
                    with respect to in-situ temperature

EXAMPLE:

t = [-10.7856; -13.4329; -12.8103; -12.2600;  -10.8863;  -8.4036;]
p = [      10;       50;      125;      250;       600;     1000;]
alpha_wrt_t_ice = gsw_alpha_wrt_t_ice(t,p)
alpha_wrt_t_ice =
  1.0e-003 *
   0.154472408751279
   0.153041866100900
   0.153232698269327
   0.153297634665747
   0.153387461617896
   0.153938395452558

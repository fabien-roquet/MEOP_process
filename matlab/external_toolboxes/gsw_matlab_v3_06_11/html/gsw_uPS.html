USAGE:

uPS = gsw_uPS

DESCRIPTION:

The unit conversion factor for salinities (35.16504/35) g/kg (Millero et
al., 2008).  Reference Salinity SR is uPS times Practical Salinity SP.

OUTPUT:

uPS  =  unit conversion factor for salinities                   [ g/kg ]

EXAMPLE:

uPS = gsw_uPS
uPS =
  1.004715428571429

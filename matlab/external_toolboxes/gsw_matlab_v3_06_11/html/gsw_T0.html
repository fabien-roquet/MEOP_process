USAGE:

T0 = gsw_T0

DESCRIPTION:

The Celcius zero point; 273.15 K.  That is T = t + T0 where T is the
Absolute Temperature (in degrees K) and t is temperature in degrees C.

OUTPUT:

T0  =  The Celcius zero point                                     [ K ]

EXAMPLE:

T0 = gsw_T0
T0 =
  273.15

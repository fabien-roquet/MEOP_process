USAGE:

f = gsw_f(lat)

DESCRIPTION:

Calculates the Coriolis parameter (f) defined by:
     f = 2*omega*sin(lat)
where,
     omega = 7.292115e-5   (Groten, 2004)                  [ radians/s ]
Click for a more detailed description of the
Coriolis parameter.

INPUT:

lat  =  latitude in decimal degrees north                [ -90 ... +90 ]

OUTPUT:

f    =  Coriolis parameter                                 [ radians/s ]

EXAMPLE:

lat = [ -90; -60; -30; 0;]
f = gsw_f(lat)
f =
1.0e-003 *
  -0.145842300000000
  -0.126303136746351
  -0.072921150000000
                   0

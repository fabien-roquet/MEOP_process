cellreal 

CELLREAL  Real part of each element in a cell array.

   XR=CELLREAL(X) where X is a cell array of N arrays,

       X{1}=X1, X{2}=X2,..., X{N}=XN
 
   returns the length N cell array XR with 
  
      XR(1)=REAL(X1), XR(2)=REAL(X2),..., XR(N)=REAL(XN).

   See also JCELL.

   Usage: xr=cellreal(x);

dawson 

DAWSON The Dawson function. [By P.J. Acklam]

   DAWSON is a low-level function called for testing purposes by 
   MORSEWAVE.

   Y = DAWSON(X) is Dawson's integral for each element of X.

   X must be real.
   
   Dawson's integral F(x) is defined as

     F(x) = exp(-x^2) * integral from 0 to x of exp(t^2) dt

   The Dawson function is also the Hilbert transform of the Gaussian,
   see for example Lilly and Olhede (2009).
  
   See also ERF, ERFC, DAWSONDERIV.

   'dawson --t' runs a test.

   Usage: x=dawson(x);
